Installation
============

Install latest release from PyPI (https://pypi.org/project/reda/):

.. image:: https://img.shields.io/pypi/v/reda.svg

.. code-block:: bash

  pip install reda

Install current development version from git:

.. code-block:: bash

  git clone https://github.com/geophysics-ubonn/reda
  cd reda

  # Option 1) Installing dependencies with pip
  pip install -r requirements.txt
  # Option 2) Installing dependencies with conda
  conda install --file requirements.txt

  python setup.py install
